# Mendel Lab: Simulador de Herencia Mendeliana

**Descripción:**
Una herramienta de software interactiva diseñada para modelar y predecir los principios de la genética clásica, aplicando las leyes fundamentales de la herencia genética.

**Función Principal:**
Simular cruces genéticos para calcular y visualizar automáticamente las proporciones exactas de fenotipos y genotipos resultantes a partir de los alelos ingresados por el usuario..
